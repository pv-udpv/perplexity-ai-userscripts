## Format Guide

### Sections for Each Release
- **Added** - New features
- **Changed** - Changes to existing features
- **Deprecated** - Soon-to-be removed features
- **Removed** - Removed features
- **Fixed** - Bug fixes
- **Security** - Security updates

### Example Entry
```markdown
## [1.1.0] - 2025-12-24

### Added
- New feature X
- Documentation for feature Y

### Fixed
- Issue with DOM selector timing (#42)
- Memory leak in event listeners

### Changed
- Improved error messages

### Deprecated
- Old utility function (will remove in 2.0.0)
```
