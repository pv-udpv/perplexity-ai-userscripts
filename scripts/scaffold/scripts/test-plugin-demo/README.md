# Test Plugin Demo

Auto-scaffolded userscript.


## Getting Started

1. Install dependencies: `npm install`
2. Implement features in `index.ts`
3. Run tests: `npm run test:watch -- test-plugin-demo`
4. Build: `npm run build`

## Development

```bash
# Build this plugin
npm run build

# Watch mode
npm run build:watch

# Run tests
npm run test test-plugin-demo
```
